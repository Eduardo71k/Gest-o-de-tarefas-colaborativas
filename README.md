# Sistema de Gestão de Tarefas Colaborativas

Sistema desenvolvido em C# para gerenciamento de tarefas em equipe.

## Funcionalidades

-  Cadastro de usuários e tarefas
-  Atribuição de responsáveis  
-  Controle de status (Pendente, Andamento, Concluída)
-  Sistema de notificações (Email, Push)
-  Padrões Observer e Strategy
-  Testes unitários
-  Pipeline CI automatizado

## Como Executar

1. Clone o repositório
2. Abra `TaskManagementSystem.sln` no Visual Studio
3. Pressione F5 para executar

## Executar Testes

`Test → Run All Tests` ou `Ctrl + R, A`
